1. O que são Tags

voltar

Em HTML (Hypertext Markup Language), as tags são elementos fundamentais usados para estruturar e formatar o conteúdo em uma página web. As tags são compostas por códigos entre colchetes angulares (< e >) e geralmente existem em pares, com uma tag de abertura e uma tag de fechamento. A estrutura básica de uma tag HTML é a seguinte:
<nome-da-tag>conteúdo</nome-da-tag>

<u> - Sublinhado: A tag <u> é usada para envolver texto que deve ser sublinhado. No entanto, é comum usar a tag <u> em HTML para indicar ênfase semântica, reservando o sublinhado para fins de estilo apenas. Exemplo:

Este é um texto sublinhado.

<strong> - Texto Forte: A tag <strong> é usada para envolver texto que é de grande importância ou enfatizado. Visualmente, o texto dentro de <strong> é frequentemente exibido em negrito, mas semanticamente, ela indica forte importância. Exemplo:

Este é um texto fortemente enfatizado.

É importante notar que, embora essas tags possam ser usadas para aplicar estilos visuais, a ênfase semântica é geralmente preferida. Além disso, a utilização de CSS para estilizar o texto é comum e oferece mais flexibilidade.

7. Tags <ol>, <ul> e <li>

voltar

As tags <ol>, <ul>, e <li> são usadas para criar listas em HTML. Aqui está uma breve explicação sobre cada uma delas:

<ol> - Lista Ordenada (Ordered List): A tag <ol> é usada para criar uma lista ordenada, onde cada item é numerado. Os itens da lista são definidos com a tag<li> (item de lista). Exemplo:

  1. Item 1
  2. Item 2
  3. Item 3
<ul> - Lista Não Ordenada (Unordered List): A tag <ul> é usada para criar uma lista não ordenada, onde cada item é marcado com um ponto, quadrado, círculo, ou outro marcador. Os itens da lista também são definidos com a tag <li>. Exemplo:
  • Item A
  • Item B
  • Item C
<li> - Item de Lista: A tag <li> é usada para definir cada item em uma lista, seja ela ordenada ou não ordenada. Os itens são colocados dentro das tags <ol> ou <ul>.
Exemplo de lista ordenada:
  1. Item 1
  2. Item 2
  3. Item 3
Exemplo de lista não ordenada:
  • Item A
  • Item B
  • Item C
Lembre-se de que essas tags podem ser aninhadas para criar listas mais complexas.

8. A tag <a>

A tag <a> em HTML é usada para criar links ou âncoras em uma página da web. Ela permite que os usuários cliquem no texto ou em um elemento e sejam redirecionados para outra página, recurso ou posição na mesma página.
Aqui está um exemplo básico de como a tag <a> é usada para criar um link:

Clique aqui para visitar o site do google.com

Neste exemplo, a tag <a> envolve o texto "Clique aqui para visitar o Exemplo.com", e o atributo href especifica o URL para o qual o link deve apontar. Ao clicar nesse link, o navegador geralmente redirecionará o usuário para o URL especificado. É possível usar a tag <a> para criar links para outros recursos, como imagens, documentos, endereços de e-mail, ou até mesmo para navegar para seções específicas de uma página usando âncoras internas. Veja alguns exemplos:

Exemplo do código html para um link de arquivo local:

<a href="documento.pdf">Baixar o documento PDF</a>

Exemplo do código html para um link de arquivo local:

<a href="mailto:contato@exemplo.com">Enviar e-mail para Contato</a>

Exemplo do código html para âncora:

<a href="#secao1">Ir para a Seção 1</a>

<h2 id="secao1">Seção 1</h2>

Esses são apenas alguns exemplos básicos. A tag <a> é fundamental para a navegação e interatividade em páginas da web.

10. Tags <sub> e <sup>

voltar

As tags <sub> e <sup> em HTML são utilizadas para definir texto como subscrito e sobrescrito, respectivamente. Aqui está uma explicação de cada uma delas:

<sub> - Subscrito: A tag <sub> é usada para representar texto que deve ser exibido como subscrito. Normalmente, ela é usada para fórmulas químicas, fórmulas matemáticas ou qualquer conteúdo onde o texto inferior é apropriado. Exemplo:

H<sub>2</sub>O
H2O

Isso renderizará "H₂O", onde o "2" está subscrito.

<sup> - Sobrescrito: A tag <sup> é usada para representar texto que deve ser exibido como sobrescrito. Geralmente, é utilizado para expoentes em expressões matemáticas ou para notações de ordem. Exemplo:

x<sup>2</sup>
x2

Isso renderizará "x²", onde o "2" está sobrescrito.

Ambas as tags são úteis para melhorar a apresentação de informações técnica e científica em uma página da web. Elas não afetam o significado semântico do texto, apenas sua apresentação visual. Lembre-se de que a aparência exata pode variar dependendo do navegador e das folhas de estilo CSS aplicadas ao documento.

11. Tag <blockquote>

voltar

A tag <blockquote> em HTML é usada para representar um bloco de texto que é uma citação de outro recurso, como um livro, um artigo ou um site. A utilização desta tag indica ao navegador que o texto contido é uma citação e geralmente resulta em um recuo ou formatação especial para destacar visualmente a citação. Aqui está um exemplo básico de como a tag <blockquote> pode ser usada:

<blockquote>
<p>Esta é uma citação de algum texto. Pode ser de um livro, artigo ou qualquer outra fonte.</p>
<footer>Nome do Autor</footer>
</blockquote>

Esta é uma citação de algum texto. Pode ser de um livro, artigo ou qualquer outra fonte.

Nome do Autor

Dentro da tag <blockquote>, você pode incluir parágrafos, cabeçalhos, listas, e outros elementos HTML para estruturar a citação. Além disso, é comum usar a tag <footer> dentro de <blockquote> para indicar o autor da citação. Este é apenas um exemplo básico, e a tag <blockquote> pode ser estilizada usando CSS para atender às necessidades de design específicas de uma página. O uso adequado da tag <blockquote> ajuda na organização e na apresentação de citações em páginas da web.
